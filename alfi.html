Penjelasan Version Control

Version Control Adalah Sistem yang digunakan untuk mengelola dan melacak perubahan pada file atau kode program dari waktu ke waktu. Dengan version control, setiap perubahan yang dilakukan pada sebuah file akan disimpan sehingga kita bisa melihat riwayat perubahan tersebut, kembali ke versi sebelumnya, atau mengetahui siapa yang melakukan perubahan.

Perbedaan Git dan Github

Git

Git adalah sistem kontrol versi yang digunakan untuk mengelola dan melacak perubahan pada file, terutama kode program.

Github

Github adalah platform berbasis web yang digunakan untuk menyimpan repository Git secara online.

Penjelasan Html

Html adalah bahasa markup standar yang digunakan untuk membuat dan menyusun halaman web. HTML digunakan untuk menentukan struktur dasar sebuah halaman website, seperti judul, paragraf, gambar, tabel, dan link.
